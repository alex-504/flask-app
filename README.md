**Running app:**
- Create the environment: 
  ```
  $ mkdir myproject
  $ cd myproject
  $ python3 -m venv .venv
  ```
- Activate the Environment:
```
  $ . .venv/bin/activate
```
- Install Flask in the Environment:
```
  $ pip install Flask
```
- Run the app:
```
  $ flask --app hello run
  * Serving Flask app 'hello'
  * Running on http://127.0.0.1:5000 (Press CTRL+C to quit)
```
